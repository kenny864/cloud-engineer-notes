# Exploring Linux system logs

## Where Linux stores logs
- /var/log

## Why logs are critical for debugging
- logs record what is happening in the system
- can use the information to troubleshoot

## Files you explored

### auth.log
- shows login attemps and ssh activity

### nginx/access.log
- shows connection activity

### nginx/error.log
- shows errors with nginx

## One thing you noticed about log format
- there is a timestamp short message explaining the error
