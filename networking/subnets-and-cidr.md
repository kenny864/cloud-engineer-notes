# subnets and cidr

## what is subnet?
- a smaller network inside a larger network

### vpc
- a network that contains a collection of subnets in

### public subnet
- connected directly to the internet
- typically bigger in size
- used for load balancer or public servers 

### private subnet
- not connected directly to the internet
- typically smaller in size 
- used for databases and internal apps

## What /24 means
- 24 bits are used for the ip range
- 8 bits are used for the subnet
- 256 different ip addresses on the subnet?

## Why cloud environments use subnets
- smaller, segmented IP address ranges used to organize, isolate, and secure cloud resources
